* Bubble Sort 
* Insertion Sort 
* Selection Sort 
* Merge Sort 
* Quick Sort 
* Heap Sort 
* Counting Sort

## Built With:
* Python 3.7.7
* Tkinter 8.6

## Run It (Python >= 3.5):
- Clone the repository
```bash
$ git clone https://github.com/FahadulShadhin/Sorting-Algorithms-Visualizer.git
```
- Run ```main.py``` from root
```bash
$ python3 main.py 
```
